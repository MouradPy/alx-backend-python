# Python Generators - Airbnb Clone Project

## Project Description
This project demonstrates advanced usage of Python generators to efficiently handle large datasets, process data in batches, and simulate real-world scenarios involving live updates and memory-efficient computations.

## Learning Objectives
- Master Python Generators and `yield` keyword
- Handle large datasets efficiently using batch processing
- Implement memory-efficient data streaming
- Integrate SQL with Python generators
- Optimize performance in data-driven applications

## Features
- MySQL database setup with ALX_prodev database
- Python generators implementation with yield keyword
- Batch processing for memory-efficient data streaming
- CSV data import functionality
- Memory-efficient data processing

## Files
- `seed.py` - Main implementation with Python generators
- `user_data.csv` - Sample user data
- `test_mysql_fixed.py` - MySQL connection test
